# Schema Rich Snippets

A Framer plugin for adding Schema.org / JSON-LD structured data to your pages for SEO rich snippets.

## Features

- 13 schema types: Article, Product, FAQ, HowTo, LocalBusiness, Organization, Person, Event, Recipe, Review, BreadcrumbList, VideoObject, SoftwareApplication
- Field-level validation with required/recommended indicators
- Live JSON-LD preview
- Google Rich Results Test integration
- Copy/embed structured data code
- Multi-schema support (multiple schemas per page)
- Preset templates for common use cases

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
